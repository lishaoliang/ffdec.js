# ffdec.js

wasm, ffmpeg, webgl

基于Emscripten的


```
文档:
https://emscripten.org/index.html

安装emcc:
https://emscripten.org/docs/getting_started/downloads.html

安装uglifyjs:
sudo npm install -g uglify-js
```

## 编译
```
cd ./ffdec.js
make
```

## 库文件
```
./ffdec.js/lib/ffdec-1.0.0.wasm
./ffdec.js/lib/ffdec-1.0.0.min.js
```
